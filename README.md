# Computer Vision from Scratch - Hands-On Implementation

This is my **hands-on learning repository** where I'm implementing all the concepts and techniques from the **[Computer Vision from Scratch](https://www.youtube.com/playlist?list=PLPTV0NXA_ZSgmWYoSpY_2EJzPJjkke4Az)** video series by [Vizuara AI Labs](https://vizuara.ai/), created by [Dr. Sreedath Panat](https://www.sreedath.com/) (MIT PhD).

## 🎯 Project Goal

This repository documents my learning journey as I master computer vision from the ground up. Rather than just watching tutorials, I'm implementing every component from first principles - from image filtering to deep neural networks - to deeply understand how modern computer vision systems work.

**Philosophy:** Build, learn, understand, and share! 🚀

## 📚 Series Overview

This 16-week comprehensive course covers the complete evolution of computer vision:

**Part 1: Foundations & Classical Computer Vision**
- Image representation and pixel manipulation
- Classical filters and convolution operations
- Edge detection (Sobel, Canny, Laplacian)
- Thresholding and morphological operations
- Feature detection (Harris corners, SIFT, SURF)
- Template matching and basic object detection

**Part 2: Machine Learning for Vision**
- Feature extraction and descriptors
- Traditional ML classifiers (SVM, KNN)
- Bag of Visual Words
- Introduction to neural networks

**Part 3: Deep Learning - CNNs**
- Convolutional Neural Networks fundamentals
- Backpropagation and optimization
- Popular architectures (AlexNet, VGG, ResNet)
- Image classification tasks
- Transfer learning applications

**Part 4: Advanced Deep Learning**
- Object detection (R-CNN, YOLO, SSD)
- Semantic segmentation
- Instance segmentation (Mask R-CNN)
- Attention mechanisms and Vision Transformers
- Video processing and understanding

**Part 5: Modern Vision Systems**
- Vision Transformers (ViT)
- Transformer-based detection and segmentation
- Multi-modal models (Vision-Language)
- State-of-the-art applications

## 💻 Implementation Philosophy

My code follows these principles:
- **From First Principles**: Building with understanding, gradually incorporating efficient libraries
- **Well-Documented**: Detailed comments explaining the "why" and "how"
- **Educational**: Type hints, docstrings, and mathematical explanations
- **Modular**: Each component is isolated and reusable
- **Practical**: NumPy and PyTorch based, balancing clarity with performance
- **Tested**: Verification on standard datasets and benchmarks

## 💡 Key Concepts Covered

- **Image Filtering**: Convolution, blur, edge detection, morphological operations
- **Classical Features**: SIFT, SURF, Harris corners, feature matching
- **Neural Networks**: Forward/backward propagation, optimization, architecture design
- **Convolutional Networks**: Layers, pooling, activation, skip connections
- **Object Detection**: Localization, classification, anchor-based and anchor-free methods
- **Segmentation**: Pixel-level classification, semantic, instance, and panoptic
- **Transformers**: Self-attention, vision transformers, multi-scale processing
- **Applications**: Real-world use cases and practical implementations
